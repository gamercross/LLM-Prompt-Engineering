## 6. 최종 선정

## 8. Prompt 개선

### v1

```text
회의록을 요약해줘
```

문제점

- 누락 발생
- 형식 일정하지 않음

---

### v2

```text
1. 결정사항

2. Action Items

3. 리스크

4. 부족하면 질문

5. 마지막에는 핵심 근거 3개
```

개선점

- 출력 통일
- 정확성 향상
- 누락 감소
